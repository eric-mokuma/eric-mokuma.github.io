CSS

Cascading style sheets, or CSS, are used to style and beautify webpages

Some CSS conceptes:

  • Margin: Margin is used to create space around elements that do not have defined borders.
  • Border: The CSS border properties allow you to specify the style, width, and color of an element's border
  • Padding: Padding is the space between the content and the border of an element.
  • Class: CSS Classes Overview, allows simultaneous styling of multiple HTML elements.
  • Ids: The id selector is utilized to choose a single unique element on a page since the id of an element is exclusive to that page. By using the id selector, you can specifically target a singular element on a page due to the uniqueness of its id within that page.
  • positions: The position property in CSS determines where an element is located within a document.
  • display: 1. The display CSS property determines how an element is treated, either as a block or inline box, and also specifies the layout for its children, such as flow layout, grid, or flex.
